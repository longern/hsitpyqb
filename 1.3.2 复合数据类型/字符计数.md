输入一个字符串，统计字符串中`s`各个字符出现了几次。

例如输入字符串：
```input
seveneleven
```
则输出为：
```output
s 1
e 5
v 2
n 2
l 1
```

```py
s = input()
count = {}
for ch in s:
    if ch not in count:
        ___1___
    ___2___
for key, value in count.items():
    print(key, value)
```
