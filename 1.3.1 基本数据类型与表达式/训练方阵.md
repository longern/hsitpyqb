某班有n名同学，训练时需要1名领队站在队伍外，其余同学组成每排k人的队伍（最后一排可能不足k人）。求队伍有几排，最后一排有几人。

```py
n = int(input())  # 总人数
k = int(input())  # 每排人数
num_rows = ___1___
remain = ___2___
print(num_rows)
print(remain)
```

例如当总人数和每排人数分别为
```input
50
8
```
时，除领队外有49人，每排8人，前6排站满8人，最后1排剩1人，因此输出为
```output
7
1
```
